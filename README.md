# 대구 미세먼지 현황 대시보드

> index.html + style.css 두 파일로 대구 대기질 대시보드 웹페이지 만들기

## 사용데이터 출처
- 공공데이터 포털(data.go.kr)
- https://www.data.go.kr/data/15159598/openapi.do#/API%20%EB%AA%A9%EB%A1%9D/v2

|항목|내용|

|API명|한국환경공단 에어코리아 대기질 정보|
